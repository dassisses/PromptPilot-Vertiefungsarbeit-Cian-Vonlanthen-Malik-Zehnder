# PromptPilot

Dieser Ordner enthält die PromptPilot-Desktop-App. Ausführliche Informationen findest du in [docs/README.md](docs/README.md).

## Schnellstart

```bash
# Abhängigkeiten installieren
./scripts/install.sh

# Anwendung starten
./scripts/start.sh  # oder python3 main.py
```

Weitere Hilfsskripte befinden sich im Ordner [`scripts/`](scripts/).
